# FlowCatalyst Website

Static website for FlowCatalyst - a toolkit for building event-driven business applications and distributed solutions.

## Development

```bash
# Start development server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```

## Features

- Responsive design with Tailwind CSS
- Optimized for hosting on static site platforms
- Mobile-friendly navigation
- Smooth scrolling and interactions
- SEO optimized

## Deployment

The website can be deployed to any static hosting service:

- Netlify
- Vercel
- GitHub Pages
- AWS S3 + CloudFront
- Any CDN or static file server

Simply upload the files or connect your repository for automatic deployments.
